# Summer Internship

A collection of small frontend examples and UI experiments (HTML/CSS/JavaScript) created for a summer internship.

##  Summary

This repository contains multiple small self-contained example projects demonstrating UI widgets, clocks, sliders, popups, and other JavaScript/CSS interactions. Each example is standalone — to run one, open its `index.html` in a browser or serve the folder with a static file server / Live Server extension.

**Structure**

- `clock/` : Simple analog/digital clock demo.
	- Files: `index.html`, `script.js`, `styles.css`

- `contact_page/` : A sample contact page layout using HTML/CSS.
	- Files: `index.html`, `styles.css`

- `count_up/` : A count-up timer or number animation demo.
	- Files: `index.html`, `css/styles.css`

- `gradient_clock/` : Clock that combines gradients and time-based styling.
	- Files: `index.html`, `script.js`, `styles.css`

- `js_slideshow/` : JavaScript-powered slideshow/carousel example.
	- Files: `index.html`, `css/styles.css`, `js/script.js`

- `jscolorgenerator/` : Color generator using JavaScript to produce palettes.
	- Files: `index.html`, `script.js`, `styles.css`

- `pop_up/` : Example of modal/pop-up functionality.
	- Files: `index.html`, `css/styles.css`

- `random_quote-generator/` : Simple random quote generator UI.
	- Files: `index.html`, `styles.css`

- `slider/` : A slider control example (image or content slider).
	- Files: `index.html`, `script.js`, `styles.css`

## How to run

- Option 1 — Open locally: In Finder (or your file manager) open a folder (for example `clock/`) and double-click `index.html` to open in your default browser.
- Option 2 — Use VS Code Live Server: Install the Live Server extension, open the workspace, right-click an `index.html` and choose "Open with Live Server".

## Notes

- All projects are client-side only (no build step or backend required).
- If a demo depends on external resources (fonts, icons), they are included or referenced in the individual folder.
- For development, use your browser DevTools to inspect and tweak styles or JavaScript.

## Contributing

- Feel free to add improvements, additional demos, or documentation. When adding a new example, create a new folder with a clear name and a working `index.html`.
